## Requisitos

 - Node 8
 - Sass
 - Chrome extension / Redux DevTools

## Instalacion

    cd front
    npm install
    cd ..
    cd back 
    npm install

## Instrucciones

### Desde /back

    npm start

EP:
http://localhost:3001/api/items/:item
http://localhost:3001/api/items/?q=:q


### Desde /front

    npm start

Entrar en http://localhost:3000
